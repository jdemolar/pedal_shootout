# #1 Pedal Specs to Include

The pedals table must include the following attributes:

* digital/analog
* true bypass/buffered
* input impedance
* output impedance
* battery capable
* power outputs (list of outputs with voltage, current, and polarity)
* preset capable (number of presets available)
* midi features (input connector, output connector, thru connector, send capabilities, receive capabilities)
* software editor available (including what platforms they are available on and what type of connections it supports, such as USB or bluetooth)
* auxiliary jacks (connector types, capabilities)
